# Build Progress: Fix User Authentication System
# Spec ID: 003-fix-user-authentication-system
# Started: 2026-01-08

## Summary
Resolve the LoginServlet authentication issue where custom user accounts fail to authenticate.
The root causes are:
1. LocalLoginServlet queries incorrect column names (doesn't match database schema)
2. Password hash format incompatibility (Java Base64 vs MySQL hex)
3. LocalLoginServlet is not mapped in web.xml (original LoginServlet is still active)

## Root Cause Analysis
- LocalLoginServlet.java queries for: password_hash, is_admin, organization_name, auth_token
- Database schema has: password, role, company_name (no auth_token)
- MySQL SHA2() produces 64-char hex string
- Java code uses Base64.getEncoder() which produces different format

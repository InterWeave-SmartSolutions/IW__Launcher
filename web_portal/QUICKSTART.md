# InterWeave Web Portal - Quick Start

## Start the Portal

**Windows:** Double-click `start_web_portal.bat`

**Linux/Mac:** 
```bash
./start_web_portal.sh
```

## Access

Open browser to: **http://localhost:8080/iw-business-daemon/IWLogin.jsp**

## Stop the Portal

**Windows:** Double-click `stop_web_portal.bat`

**Linux/Mac:**
```bash
./stop_web_portal.sh
```

## Requirements

- **Windows**: None (uses bundled Java)
- **Linux/Mac**: Java 8+ installed

================================================================================
                    IW_IDE ADMIN ACCOUNT CREDENTIALS
                           (Option 1 - Admin Only)
================================================================================

These credentials bypass the hosted database and provide direct admin access
to the Integration Manager configuration interface.

ADMIN ACCOUNTS
--------------

| Username         | Password   | Access Level | Redirect Page    |
|------------------|------------|--------------|------------------|
| __iw_admin__     | %iwps%     | Full Admin   | IMConfigA.jsp    |
| __su_admin__     | %sups%     | Super Admin  | IMConfigA.jsp    |
| __iw_support__   | $#sprt#$   | Support      | IMConfigA.jsp    |
| __iw_viewer__    | $#view#$   | View Only    | IMConfigA.jsp    |

USAGE
-----
1. Navigate to: http://localhost:8080/iw-business-daemon/IWLogin.jsp
2. Enter username and password from table above
3. You'll be redirected to the Admin Configuration page

BRANDED ADMIN ACCESS
--------------------
For portal-branded installations, use: _<brand>_
Password is the hashCode of the brand name (first 3 chars moved to end)

Example for brand "ABC":
  Username: _ABC_
  Password: hashCode("CABC") as string

CAPABILITIES
------------
- Configure transaction flows
- Set up data mappings
- Manage transformation server connections
- View/edit business daemon settings

LIMITATIONS
-----------
- No user-level access control
- No company-specific profiles
- CompanyConfiguration.jsp pages won't work
- Not suitable for multi-user environments

================================================================================
                           SECURITY WARNING
================================================================================
These are hardcoded credentials in the application bytecode. In a production
environment, consider implementing proper authentication via LocalLoginServlet
with PostgreSQL (Option 3).
================================================================================

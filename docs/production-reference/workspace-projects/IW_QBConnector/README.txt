InterWeave QuickBooks Connector
=================================

All the files in the package ZIP should be unzipped under "C:\Program Files";
don't forget to check the "Use folder names" checkbox in the WinZip extraction
dialog.

Once extracted, open the "C:\Program Files\IW_QBConnector" folder in Windows
Explorer and double-click on the "RunMe.reg" file. When prompted, click "Yes"
as necessary. There should be a success message shown.

Next, double-click on the "connector.exe" file to start the component. It will
also be automatically started every time the current user logs on. You should
see a new tray icon in the taskbar - that means that the installation finished
correctly.

For questions and further instructions please contact your InterWeave support
representative.

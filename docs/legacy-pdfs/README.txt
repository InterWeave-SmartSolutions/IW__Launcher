QUICK START
-----------
1. Double-click: start_all.bat
2. Wait 15-30 seconds
3. Open browser to: http://localhost:8080/iw-business-daemon/IWLogin.jsp
4. Login with:  __iw_admin__ / %iwps%

No Java installation required - everything is bundled!

AVAILABLE LAUNCHERS
-------------------
  start_all.bat       - Start Web Portal + IDE (recommended)
  start_ide.bat       - Start IDE only
  start_webportal.bat - Start Web Portal only
  stop_webportal.bat  - Stop Web Portal
  iw_ide.exe          - IDE application (direct)

DOCUMENTATION
-------------
  docs\WINDOWS_QUICK_START.txt           - Detailed Windows guide
  docs\authentication\README.txt         - Authentication options
  docs\authentication\ADMIN_CREDENTIALS.txt - All admin passwords

REQUIREMENTS
------------
  - Windows 7 or later
  - 512 MB RAM minimum
  - Port 8080 available
